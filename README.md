# tetris.js

tetris.js - 一款 JavaScript 开发的仿 FC 经典俄罗斯方块游戏

![俄罗斯方块游戏截图](./img/screen-shot.png)

## Features

- 支持选择难度等级：1-9级（最高等级：10级）
- 支持暂停/继续游戏
- 支持播放/暂停背景音乐
- 支持强制结束游戏
- 支持重新开始游戏
- 支持本地缓存记录最高分数
- 支持预览下一个方块

## Game Controls

- Enter：开始游戏
- ↑：转动方块
- ← →：移动方块
- ↓：方块加速向下
- SPACE：方块迅速下落
- M：暂停/继续播放背景音乐
- P：暂停/继续游戏（初始界面：长按3秒，直接从5级开始）
- Q: 结束游戏
- R: 重新开始游戏
